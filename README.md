## Install

```console
npm i @iamnapo/tsconfig -D
```

Then, add this to your `tsconfig.json`:

```js
{
	"extends": "@iamnapo/tsconfig/tsconfig.json",
	"compilerOptions": {
		"outDir": "dist"
	}
}
```
